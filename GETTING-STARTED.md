# Getting Started with GEX

Your new lead management command center. Here's how to use it.

## 🚀 First Thing Tomorrow Morning

```bash
cd ~/clawd/domain-health-dashboard

# Option 1: Visual Dashboard
gex dashboard --open

# Option 2: Terminal
gex routine
```

## 📋 Your New Daily Workflow

### 1. Morning Check (2 min)
```bash
gex routine
```
Shows overnight activity, top 5 priorities, what to focus on.

### 2. Send Emails (5-10 min)
```bash
gex queue
```
Shows ready-to-send emails with drafted responses. Copy → Paste → Send.

### 3. Mark as Done (30 sec)
```bash
gex qm done 1,2,3,4,5
```
Marks the leads you just emailed as contacted.

### 4. Quick Status Check (anytime)
```bash
gex qs
```
One-liner: `🔴 1 HOT | 1 warm | 124 pending → Run: gex queue`

## 🖥️ Visual Dashboard

Open in browser:
```bash
gex dashboard --open
```

**Keyboard shortcuts:**
- `/` - Command palette (search all commands)
- `R` - Refresh
- `Q` - Copy `gex queue`
- `?` - Show all shortcuts
