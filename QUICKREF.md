## 📅 Daily Workflow
```bash
# Morning routine
node gex.js today      # Quick overview
node gex.js nba        # Next best action
node gex.js inbox      # Priority inbox

# During the day
node gex.js fast       # Hot leads

node gex.js recent     # What happened today
```
